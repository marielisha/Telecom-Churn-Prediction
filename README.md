# Customer Churn Prediction

This repo contains the Customer Churn Prediction project as part of my data science portfolio. This project is completed as part of the Final Assignment Project in the Data Science GCI World 2026 organized by the uTokyo Matsuo Lab. A python based machine learning project developed using Google Colab that predicts customers likely to churn using customer behavioral and demographic data. Evaluation metric of the Final Assignment is accuracy i.e. percentage of telecom customer churn that is correctly predicted.

## Overview

This project applies data preprocessing, exploratory data analysis, feature engineering, and supervised machine learning techniques to identify customers who are likely to leave a service or business.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection and engineering
- Machine Learning model training
- Model evaluation using classification metrics
- Customer churn prediction

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
